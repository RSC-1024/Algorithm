## 方法

> 1. 自己讲一遍思路
> 2. 涉及到的题要写出来
> 3. 每个题就2、3点，提炼出来
> 4. 针对在纸上写的场景，把每个题类型手写2道就明白了；如果题比较大，G（），S() 可以先写主思路，然后跟面试官聊需要具体写哪部分。
